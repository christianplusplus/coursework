This is a python 3.0 script.
python 3.0 is installed on all WINDOWS lab computers.

Concatinates all of the .tst files into a single new .tst file.

To run this:
1. Put testMaker.py into the same directory as all of the tst files.
2. Open the same directory in the command line.
3. Enter "python testMaker.py".
